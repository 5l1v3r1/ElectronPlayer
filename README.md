# Netflix-Desktop
A Desktop App For Netflix Wirtten Using Electron.

# How To Use
```https://github.com/oscartbeaumont/Netflix-Electron.git
cd Netflix-Electron/
npm install
npm run dist```
Then The Exported Mac App And Linux File Will Be /dist Folder.
